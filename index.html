win10用netsh命令修改DNS

发表于 2020-05-24 分类于 电脑日常抽风

选择DNS

  • Google 的 8.8.8.8 / 8.8.4.4 (国外)
  • OpenDNS 的 208.67.222.222 / 208.67.220.220 (国外)
  • 阿里云 的 223.5.5.5 / 223.6.6.6 (国内)

    设置首选DNS

  • net_name:网络链接里显示的名称，如WLAN
  • dns_addr:你想要改成的dns地址
    netsh interface ipv4 set dns “net_name” dns_addr

    设置备用DNS

    netsh interface ipv4 add dns “net_name” dns_addr

linux起步

发表于 2020-04-23 更新于 2020-04-24 分类于 Linux

  1. Linux内打开Terminal：Ctrl+Alt+T

  2. 安装rpm包报错

    alex@ubuntu:~$ rpm -vhU xxxx.rpm
    rpm: RPM should not be used directly install RPM packages, use Alien instead!
    rpm: However assuming you know what you are doing…
    Retrieving xxxx.rpm
    error: skipping xxxx.rpm - transfer failed

  3. software updater中换下载镜像源至aliyun

  4. 简单来说，Ubuntu的软件包格式是deb，如果要安装rpm的包，则要先用alien把rpm转换成deb。
    sudo apt-get install alien #alien默认没有安装，所以首先要安装它
    sudo alien xxxx.rpm #将rpm转换位deb，完成后会生成一个同名的xxxx.deb
    sudo dpkg -i xxxx.deb #安装
